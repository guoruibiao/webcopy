# webcopy
 
 - [x] 网页版多图展示，一键生成全屏截图
 - [x] 跨平台共享剪切板
 - [ ] 剪切板支持多用户


## 部署与访问

部署
```
git clone git@github.com:guoruibiao/webcopy.git
cd webcopy/
make
```

访问
```
1. 图
    http://host:8001/image/index

2. 文
    http://host:8001/text/index

```
